## Executar o projeto

```bash
$ git clone https://github.com/Samaraferreira/api-linhas-digitaveis && cd api-linhas-digitaveis

$ npm install or yarn

$ npm start or yarn start

$ npm run test or yarn test

$ GET: http://localhost:8080/boleto/:linha
  exemple: http://localhost:8080/boleto/03399.19771 94600.990108 01534.901044 5 84380000065620
```
